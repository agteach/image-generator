Alpaca Image Generator Website

9,963 developers have joined this project

Project Get Started Discussion (72) Solutions (262)

What You Will Practice

You will learn how to implement the front end of an image generator using JavaScript, HTML, and CSS. The skills you learn in this project will help you create image generator websites where users can generate, combine, and download images.

This project should take you approximately 8 hours to complete.

Requirements

  • A website to customize your own alpaca profile picture or image
  • A customization panel for switching visual parts
  • A randomize feature for quick experiments
  • A download feature for the generated image

The demo GIF in the original brief shows one approach, but you can interpret the UI in your own way.

Suggested Implementation

  • Prepare the image assets and layer order first
  • Set up your preferred front-end stack
  • Create an alpaca preview UI
  • Add buttons for customizing each part of the image
  • Support randomized variations and downloading

References

Make sure the layer order stays correct so details like the mouth are not hidden by the face or accessories.
